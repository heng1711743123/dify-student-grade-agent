# MySQL 数据库教程

本教程基于您提供的 `student_grades` 表的创建 SQL 语句和插入 100 条数据的 INSERT 语句，逐步指导您如何在 MySQL 数据库中操作。假设您已经安装了 MySQL（如果没有，可以从 [MySQL 官网](https://dev.mysql.com/downloads/) 下载社区版）。我们将使用命令行工具（MySQL Shell 或终端）进行操作，这是最常见的入门方式。

教程分为几个部分：登录数据库、创建数据库和表、添加数据，以及一些常见问题排查。所有 SQL 语句都直接复制自您提供的代码，确保准确性。

## 步骤 1: 安装和准备 MySQL（可选，如果已安装跳过）
- **Windows/Mac/Linux**：下载 MySQL Community Server 并安装。安装时设置 root 用户密码。
- **验证安装**：打开终端（Command Prompt 或 Terminal），输入 `mysql --version` 检查是否安装成功。如果未安装，参考官方文档安装。
- **启动 MySQL 服务**：在终端运行 `sudo service mysql start`（Linux/Mac）或通过 Windows 服务管理器启动。

## 步骤 2: 登录 MySQL 数据库
1. 打开终端或命令提示符。
2. 使用以下命令登录 MySQL（替换 `<your_password>` 为您的 root 用户密码）：
   ```properties
   本地操作: mysql -u root -p
   远程服务器: mysql -u root -h 192.168.1.100 -p
   ```
   - `-u root`：指定用户名（默认是 root）。
   - `-p`：提示输入密码。
   - `-h`:远程服务器 IP 地址

3. 输入密码后，按 Enter。如果成功，您会看到 MySQL 提示符：`mysql>`。
   - 如果是其他用户，替换为 `-u your_username -p`。
   - 常见错误：
     - "Access denied"：密码错误，重置密码（参考 MySQL 文档）。
     - "Command not found"：MySQL 未添加到 PATH 环境变量，需配置。

   示例输出：
   ```
   Enter password: ******
   Welcome to the MySQL monitor.  Commands end with ; or \g.
   mysql>
   ```

## 步骤 3: 创建数据库（如果需要新数据库）
1. 在 MySQL 提示符下，创建数据库（例如命名为 `dify_test`）：
   ```sql
   CREATE DATABASE dify_test;
   ```
2. 切换到该数据库：
   ```sql
   USE dify_test;
   ```
3. 验证：运行 `SHOW DATABASES;` 查看列表。

## 步骤 4: 创建表
1. 在已切换的数据库中，执行以下 SQL 语句创建 `student_grades` 表：
   ```sql
   CREATE TABLE student_grades (
       id INT AUTO_INCREMENT PRIMARY KEY,
       student_id VARCHAR(20) NOT NULL,
       student_name VARCHAR(50) NOT NULL,
       class_name VARCHAR(50) NOT NULL,
       subject VARCHAR(50) NOT NULL,
       score DECIMAL(5,2) NOT NULL,
       exam_date DATE NOT NULL,
       semester VARCHAR(50) NOT NULL,
       grade VARCHAR(50) NOT NULL,
       created_at DATETIME NOT NULL,
       updated_at DATETIME NOT NULL
   );
   ```
2. 执行后，运行 `SHOW TABLES;` 验证表是否创建。
3. 描述表结构：`DESCRIBE student_grades;` 检查字段是否正确。
   - 说明：
     - `id` 是自增主键。
     - `score` 是小数类型（5 位总数字，2 位小数）。
     - 所有字段均为 NOT NULL，确保数据完整性。
